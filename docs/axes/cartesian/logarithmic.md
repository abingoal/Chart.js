# 对数笛卡尔轴

对数刻度用于绘制数字数据。它可以放置在x或y轴上。顾名思义，使用对数插值来确定数值在轴上的位置。

## 刻度配置选项

下列选项由对数刻度提供。它们都位于`ticks`子选项中。这些选项扩展了[常用刻度配置](README.md#tick-configuration)。

| 名称  | 类型     | 默认值 | 描述                                                                          |
| ----- | -------- | ------ | ----------------------------------------------------------------------------- |
| `min` | `Number` |        | 用户定义的最小刻度值，覆盖数据的最小值。 |
| `max` | `Number` |        | 用户定义的最大刻度值，覆盖数据的最大值。 |
